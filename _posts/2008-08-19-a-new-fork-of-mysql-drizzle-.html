Drizzle: A High-Performance Microkernel DBMS for Scale-Out Applications

A friend just clued me into this very interesting fork of MySQL, spearheaded by someone inside the MySQL team (Brian Aker) and apparently with the blessing of "Monty" Widenius

Goals include:

  • Use UTF8 and InnoDB by default
  • Rip out everything that isn't absolutely necessary, including: stored procs, views, triggers, grants
  • Make the core as small as possible
  • Sharding across multiple nodes
  • Make it a "real" community project

Neat! :)

Code is on Launchpad - it's a fork of the MySQL 6.0 codebase. They're aiming for something usable in a few months.
