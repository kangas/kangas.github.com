Installing Kismet on Nokia N810

  • Kismet - A Wi-Fi "network auditing" tool, aka passive sniffer
  • Nokia N810 - a Linux-based "internet tablet". Pocket-sized, has Wi-Fi and GPS

+

I've always wanted to put these two great tastes together. Last night I finally figured out how, courtesy of Internet Tablet Talk forums:

  • Activate RedPillMode in Application Manager, this allows you to see system libraries
  • install libraries: ncurses-base and tcpdump
  • install "becomeroot" from gronmayer.com/it so "sudo gainroot" works on the local xterm
  • Install tz1's .deb build of kismet
  • in an xterm, do:
    • sudo gainroot
    • mkdir /media/mmc11/kismet
    • kismet

And it worked! Thanks tz1!
